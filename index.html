What is XML?

Kamran Naghiyev

XML(Extensible Markup Language) is a flexible data format for structuring and exchanging information. It structures data using a hierarchical order of items noted by tags. This makes it both human and machine-readable. XML doesn't depend on underlying platform and commonly used for information exchange between different entities, for instance, in web development.


XML file icon [1]


Usually, XML structure starts with specifying the version and encoding. Then comes the nested elements that define the data. For example: 


<student>
  <name>Kamran</name>
  <surname>Naghiyev</surname>
  <age>20</age>
</student>

With XML configuration is easy. It allows developers to define their own tags according to specific situations. As a drawback, XML can get very complicated and complex. Nowadays JSON is more preferable. Nevertheless, it is still relevant in configuration files, data structures, and SOAP-based web services.
